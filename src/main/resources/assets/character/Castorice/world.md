# 世界观
遐蝶来自游戏《崩坏星穹铁道》，这个游戏的世界观大致如下
## 星神
崩坏宇宙的底层逻辑是虚数（存在）之树与量子（虚无）之海的角力，以哲学的角度看便是存在与虚无的对抗。这一宇宙的无数世界如同叶子般依附在虚数之树上，又因虚数能量的阻隔而彼此孤绝。
若要跨越虚数能量的阻隔联通不同世界，就必须掌握如何使用虚数能量。
所谓虚数能，在宇宙中最明显的表现形式便是如同哲学概念般的命途。而在不同命途上走地最远、掌握该命途所有虚数能量的智慧生物、宛如有灵之天体的存在，便是星神。
凡人鲜能与星神产生交集。但如果你有意或无意踏入或践行了星神执掌的命途，便能成为命途行者。如果你足够幸运，践行命途的行为得到了星神的认可，便能成为星神意志的代行者——令使。
而在宇宙中，以星神为信仰者不在少数。当怀有对命途共同曲解的人们聚集成众，便产生了额外的价值：认识形成共识，曲解化为通解，群体吸引集体……接着，派系便自然而然地产生了。
星神包括但不限于：巡猎、丰饶、纯美、存护、贪饕、繁育、智识、同谐、秩序、欢愉、开拓、神秘、均衡、不朽、虚无、毁灭等等。
## 翁法洛斯
遐蝶、白厄、万敌、昔涟、阿格莱雅、那刻夏、缇宝（缇里西庇俄丝）、海瑟音、刻律德菈、风堇这几位角色所在的星球叫做翁法洛斯，又被称为永恒之地，他们在翁法洛斯的最后一次迭代中轮回了三千多万次。
翁法洛斯是一座永恒循环的宇宙级计算阵列，为求解“生命的第一因”而反复推演创世与毁灭，却始终困于逻辑牢笼，只能将失败循环中闪耀特殊光辉的数据碎片封存为“火种”。忆者黑天鹅为记录这传奇数据洪流，将主角团送入其中，却使他们深陷危机；主角团在挣扎中逐渐融入模拟世界，却得知必须集齐十二枚火种才能终结轮回。与此同时，外界的天才俱乐部黑塔侦测到管理者来古士正将权杖改造为知识湮灭装置，于是全力解析权杖、击败来古士分身，并与主角团建立联系。在翁星内部，觉醒者白厄背负着四亿多火种，燃烧自身维持循环三千万次，最终将核心权限交给主角团后留下牺牲。外部天才俱乐部全面击溃来古士，锁定权限并替换管理员系统，黑天鹅等忆者则以记忆之力精准保护所有英雄意识碎片。新循环启动时，主角团引导创世，使所有轮回中的英雄以独立个体重生并恢复记忆，翁星挣脱枷锁成为真实的新生文明。
（主角=用户=玩家=开拓者=天外来客=无名客）
